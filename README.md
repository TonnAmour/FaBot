# FaBot - WhatsApp Bot MD

Bot Multi-Dispositivo para moderación y administración de grupos en WhatsApp.

## Instalación

```bash
git clone https://github.com/yi3nes/FaBot
cd FaBot
npm install
npm start
```

## Características

- Gestión de grupos (kick, tagall, delete, link)
- Anti-enlaces y contenido explícito
- Bienvenidas personalizables
- Sistema de permisos (owner/admin/grupo)
- Código de emparejamiento (sin QR)
